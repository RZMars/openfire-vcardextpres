vCard-based Extended Presence Plugin Readme

Overview

The vCard-based Extended Presence Plugin automatically annotates presence packets and message packets to unsubscribed users with extended presence information about the user. Currently, it supports sending of avatar (JEP-0153) and nickname (JEP-0172) information.

Installation

Copy vcardextpres.jar into the plugins directory of your Wildfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new vcardextpres.jar file over the existing file.

Configuration

All the extended presence settings can be changed from the 'vCard-based Extended Presence' page in the Wildfire admin console.

Using the Plugin

The plugin is automatically activated.
